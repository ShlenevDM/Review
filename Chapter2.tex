	
\subsection{Пропагатор фермиона}
	В диаграммной технике Фейнмана, которая широко применяется для расчёта различных физических величин, присутствуют внутренние линии, соответствующие виртуальным частицам или промежуточным состояниям. Пропагаторы, описывающие эти промежуточные состояния, могут содержать интересующие нас резонансы.
	
	Есть несколько вариантов представления пропагатора фермиона. Одно из них является определение в формализме собственного времени Фока~\cite{Schwinger:1951}. В этом случае пропагатор фермиона находится как решения уравнения Дирака с $\delta$-функцией в правой части или так называемой функцией Грина для уравнения Дирака:
	\begin{equation}\label{eq:Green}
		(i\partial_\mu \gamma^\mu + e_f A_\mu \gamma^\mu - m_f) S(X,X')=\delta\left(X-X'\right)\, .
	\end{equation}
Пропагатор описывается точными, но довольно громоздкими выражениями. Поэтому удобно воспользоваться различными разложениями. Например, приближение слабого магнитного поля особенно актуально для $W$-бозонов, так как критическое поле для этих частиц, равное примерно $m_W^2 / e \simeq 10^{24}$ Гс, значительно превышает магнитные поля, существующие в природе. 

С другой стороны, существование магнитных полей порядка критического значения для электрона $B \simeq B_e$ является установленным фактом. В таком случае для частиц, обладающих высоким удельным зарядом $e/m$, удобно рассматривать пропагатор в виде разложения по уровням Ландау:
\begin{equation}
	S(X,X') = \sum_{n=0}^{\infty}\sum_{s=\pm 1} S_n^s (X,X')\, .
\end{equation}

Для вычисления пропагатора также можно использовать те полевые операторы, которые были определены в разделе~\ref{Ch:WaveF}. В таком случае пропагатор будет определяться, как разность упорядоченных по времени и нормально упорядоченных произведения полевых операторов\cite{KM_Book_2013}:
\begin{equation}
	S(X,X') = T(\Psi(X)\bar{\Psi}(X')) - {\cal N}(\Psi(X)\bar{\Psi}(X'))
\end{equation}
Для дальнейших вычислений удобно будет использовать следующие волновые функции
\begin{eqnarray}
\phi^{s}_{p, \ell} (X_1) =   \frac{U^s_{\ell} [\xi(X_1)]}
{\sqrt{2 M_\ell (E_{\ell} + M_\ell)(M_\ell + m_f)}} \, .
\label{eq:phi_psi}
\end{eqnarray}
Используя точные решения уравнения Дирака \ref{eq:psie} пропагатор можно представить следующим образом:
\begin{eqnarray}\label{eq:propagator}
\hat S^{s}_n (X, X^{\,\prime}) =  \int \frac{\dd p_0 \dd p_y 
	\dd p_z}{(2\pi)^3} 
\frac{\eee^{- \, \ii \, \left( p \,(X - X') \right)_{\parallel} + 
		\ii p_y \,(X_2 - X'_2)}}
{p^{2}_{\mprl} - M_n^2 - {\cal R}^{s}_\Sigma (p) + \ii\, {\cal I}^{s}_\Sigma (p)}
\, \phi^{s}_{p, n} (X_1) \bar \phi^{s}_{p, n} (X'_1) \, ,
\end{eqnarray}
где ${\cal R}^{s}_\Sigma (p)$ и ${\cal I}^{s}_\Sigma (p)$ -- реальная и мнимая части массового оператора фермиона.  Для их получения требуется вычислить радиационные поправки к массе фермиона в замагниченной плазме. Реальная часть массового оператора ${\cal R}^{s}_\Sigma (p)$ и ${\cal I}^{s}_\Sigma (p)$ определяет изменение закона дисперсии фермиона в присутствии замагниченной плазмы. Для случая сильного магнитного поля $B\gtrsim B_e$ без учета плазмы лидирующий вклад в  сдвиг массы фермиона, находящегося на основном уровне Ландау, описывается квадратом логарифмической функции~\cite{Jancovici:1969}:
\begin{eqnarray}
\label{eq:Re2}
\Re^{s}_\Sigma (p) = \frac{\alpha}{4\pi} m_f \ln^2 (2\beta/m_f^2).
\end{eqnarray}
Откуда следует, что даже для достаточно больших значений магнитного поля $B\lesssim 10^{16}$ Гс эта добавка является несущественной и имеет порядок постоянной тонкой структуры $\alpha$.
 
Резонанс на виртуальном фермионе будет наблюдаться, когда в знаменателе пропагатора~\ref{eq:propagator} реальная часть равна нулю. Тогда виртуальная частица становится реальной, то есть приобретает определенный закон дисперсии $p_\parallel^2 - M_n^2=0$. При этом учет мнимой части массового оператора  ${\cal I}^{s}_\Sigma (p)$ вблизи резонанса становится необходимым. Она может быть получена с помощью оптической теоремы и представлена в следующем виде~\cite{MGU, Zhukovskii:1994}:
%
\begin{eqnarray}
\Im^{s}_\Sigma (p) = - \frac{1}{2}\, p_0 \; \Gamma_n^{s} \, ,
\label{eq:I_Sigma}
\end{eqnarray} 
\noindent где $\Gamma_n^{s}$ -- полная ширина поглощения фермиона, находящегося 
в поляризационном состоянии $s$ и занимающего  $n$-й уровень  Ландау. Она определяет время существования виртуальной частицы, которое в нерезонансной области предполагается бесконечно большим.
Полная ширина изменения состояния фермиона может быть выражена через 
ширину рождения фермиона~\cite{Weldon:1983}:
%
\begin{eqnarray}
\label{Gns}
\Gamma_n^{s} = \Gamma^{(abs)\, s}_n + \Gamma^{(cr) \, s}_n \simeq 
\Gamma^{(cr) \, s}_{n} 
\left [1+ \eee^{(E^{\, \prime \prime}_n - \mu)/T} \right ]
\end{eqnarray}

Таким образом введенный пропагатор позволяет корректно рассчитывать сечения в резонансной области.

\subsection{Пропагатор фотона}
Поляризационные и дисперсионные свойства фотона определяются поляризационным оператором фотона ${\cal P}_{\alpha\beta}$, явный вид которого может быть получен из результатов работ~\cite{Shabad:1988,Tsai:1974,Batalin:1971,Skobelev:1975}. Для анализа свойств поляризационного оператора удобно разложить ${\cal P}_{\alpha\beta}$ по базису из 4-векторов~\cite{Batalin:1971}, построенного из тензора электромагнитного поля и 4-вектора импульса фотона~$q_\alpha$:
\begin{equation}
\begin{gathered}
\label{eq:basis}
b_{\mu}^{(1)} = (\varphi q)_\mu, \qquad
b_{\mu}^{(2)} = (\tilde \varphi q)_\mu, 
\\
b_{\mu}^{(3)} = q^2 \, (\Lambda q)_\mu - q_\mu \, q^2_{\mbox{\tiny $\bot$}}, 
\qquad b_{\mu}^{(4)} = q_\mu, 
\end{gathered}
\end{equation}

\noindent являющихся собственными векторами поляризационного оператора в постоянном 
однородном магнитном поле. При этом $(b^{(1)} b^{*(1)}) = -q^2_{\mprp}$, 
$(b^{(2)} b^{*(2)}) = -q^2_{\mprl}$, $(b^{(3)} b^{*(3)}) = -q^2 q^2_{\mprl} 
q^2_{\mprp}$, $(b^{(4)} b^{*(4)}) = q^2$.  

Функцию Грина можно разложить  по собственным векторам $r_\alpha^{(\lambda)}$ поляризационного оператора в замагниченной плазме:
\begin{equation}\label{eq:InvGcFourier}
G_{\alpha\beta}(x)=\int \frac{\dd^4q}{(2\pi)^4}G_{\alpha \beta}(q) e^{-\ii qx}\,
\end{equation}
\begin{equation}\label{eq:GcFourier}
G_{\alpha\beta}(q)=\sum_{\lambda=1}^{3}\frac{r_\alpha^{(\lambda)}r^{(\lambda)}_\beta}{(r^{(\lambda)})^2}\cdot \frac{1}{q^2-{\cal P}^{(\lambda)}(q)}\, ,
\end{equation}

В замагниченной плазме ${\cal P}_{\alpha \beta}$ уже не будет диагональным в базисе из векторов~(\ref{eq:basis}), поэтому его удобно разложить  по собственным векторам 
$r_{\alpha}^{(\lambda)}$ в замагниченной плазме с соответствующими собственными 
значениями ${\cal P}^{(\lambda)}$~\cite{Rojas1979r,Rojas1982,Shabad:1988,MRCh:2014}:
%
\beq
\label{eq:Pab10}
{\cal P}_{\alpha \beta} = \sum_{\lambda = 1}^{3} 
{\cal P}^{(\lambda)} \, \frac{r_{\alpha}^{(\lambda)} 
	(r_{\beta}^{(\lambda)})^{*}}{(r^{(\lambda)})^2} \, , \quad 
r_{\beta}^{(\lambda)} = \sum\limits_{i = 1}^{3} A_i^{(\lambda)} \, b_{\beta}^{(i)} \, , 
\eeq
\noindent где  $A_i^{(\lambda)}$ некоторые комплексные коэффициенты.
