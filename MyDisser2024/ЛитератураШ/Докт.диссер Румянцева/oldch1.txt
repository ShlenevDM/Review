$S$-матричный элемент в древесном приближении описывается диаграммами Фейнмана, изображенными 
на рис.~\ref{fig:Diagjj} и имеет вид  

%
\beq                          
\nonumber
S^{s^{\, \prime} s}_{k^{\, \prime} k} &=& - g_k g_{k'}\int \dd^4 X \dd^4 Y j_k (X) j_{k'} (Y)
\left [\bar \Psi^{s'}_{p',\ell'}(Y) \Gamma_{k'} 
\hat S(Y,X) 
\Gamma_k \Psi^{s}_{p,\ell}(X) \right ]
\\
\label{eq:S1}
&+&  (j_k, \Gamma_k \leftrightarrow j_k', \Gamma_k')\, .
\eeq

\noindent Здесь,  $p^{\mu} = (E_{\ell}, {\bf p})$ и $p^{\, \prime \mu} =
(E^{\, \prime}_{\ell'}, {\bf p}^{\, \prime})$  -- 4-импульсы начального и конечного фермионов соответственно,  
 $\Psi^s_{p,n}(X)$ -- волновая функция фермиона в представлении внешнего магнитного поля, 
 $X^{\mu} = (X_0, X_1, X_2, X_3)$. 

Существует несколько возможных способов построения волновых функций фермионов в представлении внешнего 
магнитного поля  посредством решения уравнения Дирака  (см., 
например~\cite{Johnson:1949,Akhiezer:1965,Sokolov:1968,Melrose:1983a,Sokolov:1986,Bhattacharya:2004,Balantsev:2011} 
и также~\cite{KM_Book_2003,KM_Book_2013}).
 В большинстве случаев эти решения представлены в виде, когда две верхние компоненты биспиноров  соответствуют 
 состояниям фермиона с проекциями спина на направление магнитного поля, равными $1/2$ и   $-1/2$. 
Однако, при решении данной задачи оказалось удобным использовать другое представление волновых функций фермиона.

В работе~\cite{Sokolov:1968} был введен оператор, названный обобщенным тензором спина и являющийся тензором 
третьего ранга. В современных общепринятых обозначениях этот оператор имеет вид

%
\begin{eqnarray}
{\rm F}_{\mu \nu \lambda} = - \frac{\ii}{2} \left( P_\lambda \gamma_0 \sigma_{\mu \nu} 
+ \gamma_0 \sigma_{\mu \nu} P_\lambda \right),
\label{eq:Fgen}
\end{eqnarray}
%
\noindent где $\sigma_{\mu \nu} = (\gamma_\mu \gamma_\nu - \gamma_\nu \gamma_\mu)/2$, и  
$P^\lambda = \ii \partial^\lambda + e \, A^\lambda = \left( \ii \partial_0 + e \, A_0 \,, 
- \ii {\bs \nabla} + e {\bs A} \right)$ -- оператор обобщенного 4-импулься с 4-потенциалом внешнего магнитного поля $A^\lambda$. 
Заметим, что в работе~\cite{Sokolov:1968}, ковариантные билинейные формы были построены из матриц Дирака в обкладках  биспиноров 
$\psi^{\dagger}$ и $\psi$, тогда как в современной литературе~\cite{Peskin:1995} билинейные формы строятся из матриц 
Дирака в обкладках биспиноров $\bar\psi$ и $\psi$.  



Если взять компоненту ${\rm F}_{\mu \nu 0}$ оператора~(\ref{eq:Fgen}) и  учитывая, что  уравнение Дирака в форме  Шредингера 
имеет вид  $\ii \partial_0 = H$, где 
$H = \gamma_0 \left( {\bs \gamma} {\bs P} \right) + m_e \, \gamma_0 - e A_0$ 
 -- гамильтониан Дирака, мы можем построить векторный оператор
%
\begin{eqnarray}
{\mu}_i = - \frac{1}{2} \, \varepsilon_{ijk} \, {\rm F}_{jk0} \,, 
\label{eq:mu_i}
\end{eqnarray}
%
где $\varepsilon_{ijk}$ -- тензор Леви-Чевита. В векторной форме~(\ref{eq:mu_i}) имеет вид 
%
\begin{eqnarray}
{\bs \mu} = m_e {\bs \Sigma} - \ii \gamma_0 \gamma_5 [{\bs \Sigma} \times \hat{\bs P}] \,. 
\label{eq:mu_vec}
\end{eqnarray}
\noindent и представляет собой оператор магнитного момента~\cite{Sokolov:1968,Melrose:1983a}.

%
Нетрудно показать,  что компоненты оператора~(\ref{eq:mu_vec}) коммутируют с гамильтонианом, т.е. 
$H$ и ${\mu}_z$ имеют общие собственные функции.  В нерелятивистском пределе, оператор~(\ref{eq:mu_vec}) 
переходит в обычный оператор Паули для магнитного момента, который имеет явную физическую интерпретацию.  
 
Это позволяет использовать волновые функции фермионов как собственные 
функции оперетора ${\mu}_z$~\cite{Sokolov:1968,Melrose:1983a} 
%
\begin{eqnarray}
{\mu}_z = m_e \Sigma_z - \ii \gamma_0 \gamma_5 [{\bs \Sigma} \times {\bs P}]_z \,, 
\label{eq:mu_z}
\end{eqnarray}
%
где ${\bs P} =  - \ii {\bs \nabla} + e {\bs A}$. 




Наш выбор решений уравнения Дирака как собственных функций оператора  $\hat{\mu}_z$
можно обосновать следующими аргументами. Расчеты ширин процессов с двумя или более вершинами
во внешнем магнитном поле стандартным методом, а именно,  возведение амплитуды процесса в квадрат
с учетом всех диаграмм Фейнмана в данном порядке теории возмущений и с последующим суммированием или усреднение по  
поляризационным состояниям фермионов, представляют собой значительные вычислительные трудности. 
В этом случае, для нахождения парциальных 
вкладов в амплитуду от каналов с различными поляризационными состояниями фермионов   удобно 
проводить  вычисления 
для каждой диаграммы отдельно, путем прямого умножения биспиноров и матриц Дирака.
Результат, с точностью до общей для обеих диаграмм неинвариантной фазой, будет иметь явные 
лоренц-инвариантные структуры. Однако, амплитуды полученные с помощью решений для
фиксированного направления спина, не имеют лоренц-инвариантной структуры. 
Лоренц-инвариантной  
будет только квадрат амплитуды, просуммированный по поляризационным состояниям фермионов.

%В приложение
Мы работаем в системе отсчета, где магнитное поле направлено вдоль оси $z$ и используем калибровку 
Ландау, в которой 4-потенциал имеет вид $A^\lambda = (0, 0, x B, 0)$. Удобно использовать следующее 
обозначение $\beta = |e_f| B$ и  ввести знак заряда фермиона как $\eta = e_f/|e_f|$. Тогда волновые 
функции фермионов можно представить в  виде 
%
\beq
\label{eq:psie}
\Psi^s_{p,n}(X) = \frac{e^{-\ii(E_{n} X_0 - p_y X_2 - p_z X_3)}\; U^s_{n} (\xi)}
{\sqrt{4E_{n}M_n (E_{n} + M_n)(M_n + m_f) L_y L_z}} \, ,  
\eeq
%
где 
%
\beq
\label{eq:E_n,M_n}
E_n = \sqrt{M_n^2 + p_z^2}\, , \quad  M_n = \sqrt{m_f^2 + 2 \beta n}\, ,
\eeq
%
и являются решениями уравнения
%
\beq
\label{eq:mu_z_Eq}
\hat{\mu}_z \,\Psi^s_{p,n}(X) = s \, M_n \, \Psi^s_{p,n}(X) \, , \quad s = \pm 1\,.
\eeq
%
Удобно представить биспиноры  $U^s_{n} (\xi)$ в виде разложения по решениям для 
положительного и отрицательного заряда фермиона, $U^s_{n, \eta} (\xi)$:
%
\beq
\label{eq:U^s}
U^s_{n} (\xi) = \frac{1-\eta}{2} \, U^{s}_{n,-} (\xi) + \frac{1+\eta}{2} \, U^{s}_{n,+} (\xi) \,,
\eeq
%
где
%
\beq
\label{eq:U--}
&&U^{-}_{n,-} (\xi) = \left ( 
\begin{array}{c}
-\ii\sqrt{2\beta n} \, p_z V_{n-1} (\xi)\\[2mm]
(E_n + M_n)(M_n + m_f) V_n (\xi)\\[2mm]
-\ii\sqrt{2\beta n} (E_n + M_n) V_{n-1} (\xi)\\[2mm]
-p_z (M_n + m_f) V_n (\xi)
\end{array}
\right )  ,   
%
\\ [3mm]
\label{eq:U+-}
&&U^{+}_{n,-} (\xi) = \left ( 
\begin{array}{c}
(E_n + M_n) (M_n + m_f) V_{n-1} (\xi)\\[2mm]
-\ii\sqrt{2\beta n} \, p_z V_n (\xi)\\[2mm]
p_z (M_n + m_f) V_{n-1} (\xi)\\[2mm]
\ii \sqrt{2 \beta n} (E_n + M_n) V_n (\xi)
\end{array}
\right )\! , 
\eeq
  
%
\beq
\label{eq:U-+}
&&U^{-}_{n,+} (\xi) = \left ( 
\begin{array}{c}
\ii\sqrt{2\beta n} \, p_z V_{n} (\xi)\\[2mm]
(E_n + M_n)(M_n + m_f) V_{n-1} (\xi)\\[2mm]
\ii\sqrt{2\beta n} (E_n + M_n) V_{n} (\xi)\\[2mm]
-p_z (M_n + m_f) V_{n-1} (\xi)
\end{array}
\right ) \!\! ,   
%
\\ [3mm]
\label{eq:U++}
&&U^{+}_{n,+} (\xi) = \left ( 
\begin{array}{c}
(E_n + M_n) (M_n + m_f) V_{n} (\xi)\\[2mm]
\ii\sqrt{2\beta n} \, p_z V_{n-1} (\xi)\\[2mm]
p_z (M_n + m_f) V_{n} (\xi)\\[2mm]
-\ii \sqrt{2 \beta n} (E_n + M_n) V_{n-1} (\xi)
\end{array}
\right ) , 
\eeq
%
$V_n(\xi) \, (n = 0,1,2, \dots)$ -- нормированные функции гармонического осциллятора, которые 
следующим образом выражаются через полиномы Эрмита $H_n(\xi)$ \cite{Gradshtein}:
%
\beq
\label{eq:V_n}
V_n (\xi) = \frac{\beta^{1/4}\eee^{-\xi^2/2}}{\sqrt{2^n n! \sqrt{\pi}}} \, H_n(\xi)\, ,
\eeq
%
\beq
\label{eq:xi}
\xi = \sqrt{\beta} \left (X_1 - \eta \frac{p_y}{\beta} \right ) .
\eeq
%
Токи $j_k$ в уравнении~(\ref{eq:S1})  могут быть выражены через амплитуды в импульсном пространстве:
%
\beq
\label{eq:j_k}
j_k (X) = \frac{e^{-\ii(qX)}}{\sqrt{2q_0 V}} \, j_k(q) \,.
\eeq
