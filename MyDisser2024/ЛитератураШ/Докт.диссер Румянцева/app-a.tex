
\chapter{Результаты для ${\cal R}^{s^{\, \prime} s}_{k^{\, \prime} k}$}

\label{Sec:app_a}


Факторы ${\cal R}^{s^{\, \prime} s}_{k^{\, \prime} k}$, входящие в~(\ref{eq:M12b}), приведены для случая отрицательного 
заряда фермиона, $\eta = -1$ и  могут быть получены 
с использованием волновых функций~(\ref{eq:psie}).
%Результаты для ${\cal R}^{s^{\, \prime} s}_{k^{\, \prime} k}$ приведены для случая отрицательного 
%заряда фермиона, $\eta = -1$.

${\cal I}_{n, \ell} \equiv {\cal I}_{n, \ell}
\left ({q^{2}_{\mprp}}/{(2 \beta)} \right )$ и 
${\cal I}^{\, \prime}_{n, \ell'} \equiv {\cal I}_{n, \ell'}
\left ({q^{\, \prime 2}_{\mprp}}/{(2 \beta)} \right )$.
